# React Likert

React components for drawing various Likert visualizations in your tables. Uses D3 to draw a continuous 
chart over the column, yet leaves your table structure intact and adds aria tags for readable data in 
the original cells.

[GitHub](https://github.com/jasonphillips/react-likert) | [npm](https://www.npmjs.com/package/react-likert)

## Diverging Likert Bars
